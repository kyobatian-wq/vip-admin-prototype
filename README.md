## 📌 核心迭代内容一览

1. **总开关**：本期固定为「开启」状态并锁定，不支持关闭；点击无任何响应，二次确认弹窗设计留作后续版本预留
2. **稽核倍数**：仅支持正整数，所需稽核流水 = 奖励金额 × 稽核倍数（全局统一配置）
3. **稽核限定平台**：支持按游戏商展开勾选具体游戏
4. **VIP等级配置**：只读表格 + 编辑弹窗，弹窗内按等级区块单独 ＋/－ 新增删除
5. **保级机制**：每月1号判定上月（自然月）充值/流水，不达标降1级；晋级奖励的保留/重复领取规则受保级开关影响，日/周/月奖励领取逻辑与保级开关无关（详见PRD 5.6.1）
6. **领取规则**：4种未领取奖励处理方案，支持"N天内充值"受众判定
7. **规则说明**：默认文案 / 自定义富文本编辑器，支持背景图上传与实时预览
8. **客户端补充页面**：新增"Instruções para manter o nível"（保级说明）页签
9. **客户端领取按钮状态**：未达标按钮真正禁用（不可点击、无弹窗）；已达标按钮点击后二次判断区间内是否充值，分别弹出"充值引导"或"领取成功"弹窗
